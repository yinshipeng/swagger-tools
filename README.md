# swagger-tools

> 一个集成了swagger-ui、swagger-editor和json-server等插件，用于前端接口快速开发和测试的演示项目。

## Build Setup

``` bash
# install dependencies
npm install

# run server for swagger-ui
npm run server-ui

# run server for swagger-editor
npm run server-editor

# run json-server for mock data
npm run json-server

```
